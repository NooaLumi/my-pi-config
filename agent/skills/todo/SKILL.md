name: todo
description: Operate on the user's todo list. List-, add-, remove-, or move todo items. Use this when you want to manage the user's todos, tasks or reminders. 
--- 
# Usage

## List items

Run command:
```
{baseDir}/scripts/list_todo.sh
```

## Add item

Run command:
```
{baseDir}/scripts/add_todo.sh <todo item>
```

## Remove or move item

Remove or move the item by writing to `~/syncthing/Notes/todo.txt`
